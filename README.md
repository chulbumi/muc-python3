## 빈 폴더 작성 시
개발을 하다보면 '빈 폴더'가 필요한 경우가 있습니다. 그런데 GIT은 빈 폴더는 커밋대상으로 취급하지 않습니다.
그래서 꼼수를 이용합니다.

빈폴더 안에 별도의 .gitignore를 등록하고 해당 파일 안에 이렇게 입력합니다

```
* //이 폴더 안의 모든 것을 무시
!.gitignore // 대신 이 파일 빼고
```

이렇게하면 빈 폴더가 Commit되고, 폴더 하위에 혹시라도 발생하는 Dummy Data는 커밋대상으로 취급하지 않습니다.
