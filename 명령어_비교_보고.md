# 명령어 비교 보고서 (Python 9900 vs Rust 9999)

## 수정 완료된 사항

### 1. 소지품 명령어
- **문제**: Rust 네이티브 `inventory_command`가 Rhai 스크립트보다 우선 적용되어 다른 포맷 출력
- **해결**: `src/command/commands/equipment.rs`에서 "소지품" 명령 등록 제거
- **결과**: 이제 `cmds/소지품.rhai` 스크립트가 처리

### 2. 능력치/점수 명령어
- **문제 1**: 별칭 방향이 Python과 반대였음
- **문제 2**: Python에는 "능력치" 명령/별칭이 없음
- **해결 1**: `src/command/registry.rs`에서 "능력치" → "점수" 별칭 제거
- **해결 2**: `cmds/능력치.rhai` → `cmds/능력치.rhai.disabled`로 비활성화
- **결과**: Python과 동일하게 "능력치"는 인식 안됨, "점수"가 메인 명령

### 3. 봐 명령어
- **문제**: `starts_with` 함수 미정의로 문법 오류
- **해결**: 스크립트 간소화 및 `substring` 사용
